# Rhombixtechnologies_task3

# Task 1- Salon Booking System UI

This project is a complete UI/UX design of a Salon Booking System created in Figma for both mobile and web. It provides a smooth and user-friendly booking experience for users.

Users can browse services, select a specialist, choose a date and time, review their order, and confirm their booking.

# Platforms
Mobile Application,
Website (Web Version)

# Features 
Splash Screen,
Home Screen,
Service Details Screen,
Specialist Selection Screen,
Booking Screen,
Order Summary Screen,
Confirmation Screen
 
# Design Highlights
User-friendly interface, Consistent visual hierarchy, Simple navigation, Dark theme design, Mobile responsive layout

# Tools Used
Figma

To design an intuitive and efficient booking experience that helps users complete appointments quickly and easily.


# Task 2- Mini Design System

This project is a Mini Design System created in Figma to establish a consistent visual language across user interfaces. The design system is based on a dark-themed salon application.

It includes reusable components and style guidelines that can be applied to future UI screens.

# Components Included
Color Palette,
Typography,
Buttons,
Input Field,
Cards,
Spacing Rules

# Tool Used
Figma

To create a consistent and reusable design system that improves usability and maintains visual consistency across user interfaces.
